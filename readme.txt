WP Filter & Combine RSS Feeds is a plugin designed to manage or multiple RSS feeds with an integrated search engine. The extension is multi-site compatible and offers many filters and customization settings (WP Filter & Combine RSS Feeds est un plugin destin&eacute; &agrave; g&eacute;rer ou plusieurs flux RSS avec un moteur de recherche int&eacute;gr&eacute;. L'extension est compatible multisite et offre de nombreux filtres et param&egrave;tres de personnalisation).

== Description ==

<p><em>For English people</em></p>
<p><strong>WP Filter & Combine RSS Feeds</strong> works in multilingual and multisite<br/>
<strong>WP Filter & Combine RSS Feeds</strong> allows you to view all publications from one or more RSS feeds with or without a search filter. If multiple feeds are provided, the plugin combines the data to make one feed.<br/>
Many settings allow you to customize the experience and customize the final display of RSS feeds. You can also choose a CSS template or create your own with the CSS selectors provided in the documentation.</p>

<p><em>En fran&ccedil;ais</em></p>
<p><strong>WP Filter & Combine RSS Feeds</strong> fonctionne en multilingue et en multisite<br/>
<strong>WP Filter & Combine RSS Feeds</strong> permet d'afficher toutes publications d'un ou plusieurs flux RSS avec ou sans filtre de recherche. Si plusieurs feeds sont fournis, le plugin combine les donn&eacute;es pour ne faire plus qu'un flux.<br/>
De nombreux param&egrave;tres permettent de personnaliser l'exp&eacute;rience et de personnaliser l'affichage final des flux RSS. Vous pouvez &eacute;galement choisir un template CSS ou cr&eacute;er le v&ocirc;tre avec les s&eacute;lecteurs CSS fournis dans la documentation.
</p>

== Installation ==

<p><em>For English people</em></p>
<p>1. Copy the content of the archive in "wp-content/plugins" and activate the plugin.<br/>
2. Adjust the settings in the options.<br/>
</p>

<p><em>En fran&ccedil;ais</em></p>
<p>1. Recopier le contenu de l'archive dans le r&eacute;pertoire des extensions (wp-content/plugins) et activer dans le panneau des extensions du backoffice de Wordpress.<br/>
2. R&eacute;gler les param&egrave;tres dans les options de l'extension.<br/>
</p>
